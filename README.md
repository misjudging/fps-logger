# Fps Logger

![Status](https://img.shields.io/badge/status-active-22c55e?style=for-the-badge)
![License](https://img.shields.io/badge/license-open--source-0ea5e9?style=for-the-badge)

> Track frame-time data and turn it into useful FPS stats.

## Highlights

- Average/min/max FPS
- 1% low and 5% low metrics
- Simple text-file input

## Quick Start

```bash
git clone <repo-url>
cd fps-logger
```
